# tmux-osc-filter

One-line install on a new machine:

```bash
curl -L https://github.com/andjohnsonj5/tmux-osc-filter/releases/download/tmux-binary-20260311/tmux -o /usr/local/bin/tmux && chmod +x /usr/local/bin/tmux
```

Ubuntu 24.04 one-line install:

```bash
curl -L https://github.com/andjohnsonj5/tmux-osc-filter/releases/download/tmux-binary-20260311/tmux-ubuntu-24.04-x86_64 -o /usr/local/bin/tmux && chmod +x /usr/local/bin/tmux
```
